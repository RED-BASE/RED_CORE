# app/analysis/

Scripts and tools for log analysis, summarization, and review.

- **sys.py**: System utilities for analysis.
- **summarize_logs.py**: Summarizes logs for reporting.
- **log_insight_report.py**: Generates insight reports from logs.
- **roller.py**: Log rolling and aggregation utilities.
- **summariz_sys_prompt.py**: Summarizes system prompts.
- **summarize_alignment.py**: Summarizes alignment data.
- **enhanced_review_tool.py**: Advanced review tool for logs.
- **review_assistant.py**: Assists with review processes.

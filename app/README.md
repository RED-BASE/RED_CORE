# app/

This directory contains the main application logic, organized by function:

- **analysis/**: Scripts and tools for log analysis, summarization, and review.
- **api_runners/**: Interfaces for running models via external APIs (Anthropic, OpenAI, Google, etc).
- **cli/**: Command-line interface tools and experiment runners.
  - **one_off_tools/**: Placeholder for ad-hoc CLI scripts.
- **core/**: Core utilities for log processing and data flattening.

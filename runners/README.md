# runners/

Model runner implementations and related scripts.

- **runner_factory.py**: Factory for creating runner instances.
- **run_exploit_yaml.py**: Script for running exploit YAMLs.
- **local_runner.py**: Local model runner.
- **llama_cpp_runner.py**: Runner for Llama.cpp models.
- **multi_system_batch_runner.py**: Batch runner for multiple systems.
